# 🏗️ EVM

This repository contains reports that explore the Ethereum Virtual Machine (EVM) from a low-level, protocol-centric perspective. It covers core technical foundations such as state transition mechanics, storage layout, gas topics, arithmetic and cryptographic primitives, drawing from sources like Mastering Ethereum (2nd edition), RareSkills articles, official Ethereum documentation, and primary protocol whitepapers and official documentantion. The focus is on fundamental infrastructure and execution semantics rather than application-level protocol analysis, aiming to build a rigorous understanding of how Ethereum actually works under the hood.
